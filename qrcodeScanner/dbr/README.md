# Dynamsoft Barcode Reader for Raspberry Pi

## Getting Started
1. Build C/C++ samples:

    ```
    cd samples/c
    make
    ```
2. Read barcode images:

    ```
    ./BarcodeReaderDemo ../../AllSupportedBarcodeTypes.tif
    ```
